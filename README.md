# rollcall
- required
1.Node.JS
2.Ionic Framework
3.Cordova
4.Android Studio

- Install Project
1. Clone rollcall Project
2. Enter command "npm install" in Rollcall Project
3. Run Project with Command "ionic cordova run android --device" Or "ionic cordova emulate android"
